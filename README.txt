Descrição:
Atividade prática de Git e GitHub de versionamento.

Comandos utilizados:
- git clone: clona o repositório do GitHub
- git add: adiciona arquivos para o controle de versão
- git commit: salva as alterações localmente
- git push: envia as alterações para o GitHub
- git branch: cria uma nova ramificação
- git checkout: muda de ramificação
- git merge: mescla duas ramificações
